# P3 Moonlight — A Persona 3 inspired fcitx5 skin

A dark blue fcitx5 input method skin inspired by the UI style of *Persona 3*.

- Dark blue rectangular input panel
- Cyan rectangular highlight with white text
- Only two SVG assets: `panel.svg` and `highlight.svg`
- Original SVG assets generated by `scripts/generate_assets.py`
- Safe for open-source distribution (no official game assets included)

## Installation

```bash
# 1. Clone or download this repository

# 2. Generate the assets (optional — pre-generated assets are in dist/)
python scripts/generate_assets.py

# 3. Copy the skin to your fcitx5 themes directory
mkdir -p ~/.local/share/fcitx5/themes/
cp -r dist/p3-skin ~/.local/share/fcitx5/themes/p3-moonlight

# 4. Set the theme in fcitx5
# Option A: using fcitx5-configtool
fcitx5-configtool
#    → Addons → Classic User Interface → Theme → Select "P3 Moonlight"

# Option B: edit the config file directly
# ~/.config/fcitx5/conf/classicui.conf
# Theme=p3-moonlight
# DarkTheme=p3-moonlight

# 5. Restart fcitx5
fcitx5 -r
```

## Recommended font

The skin itself does not bundle a font. For the best look, install **M+ Fonts**:

```bash
# Fedora
sudo dnf install mplus-1c-fonts mplus-1m-fonts

# Arch Linux
sudo pacman -S otf-mplus
```

Then set it in `~/.config/fcitx5/conf/classicui.conf`:

```ini
Font="M+ 1c bold 13"
MenuFont="M+ 1c bold 13"
TrayFont="M+ 1c bold 10"
```

M+ Fonts has a Square Gothic feel that is closest to the Rodin / Skip fonts used in the official *Persona 3* UI.

## Development

Edit `scripts/generate_assets.py` to tune colors, panel size, etc. Then rerun:

```bash
python scripts/generate_assets.py
cp -r dist/p3-skin ~/.local/share/fcitx5/themes/p3-moonlight
fcitx5 -r
```
